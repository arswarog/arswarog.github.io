Алиасы в git

4 Nov 2018
tools

В GIT есть очень удобная функциональность для создания кратких команд:

1

git config --global alias.имя_алиаса команда

Например, очень удобная команда:

1

git log --oneline --graph

Создающая вывод в терминале всего дерева коммитов

Очень удобно сделать сокращение такой команды

1

git l

для этого надо выполнить

1

git config --global alias.l log --oneline --graph

или улучшенный вариант

1

git config --global alias.l log --oneline --graph --pretty=format:'%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr)%C(bold blue) <%an>%Creset' --abbrev-commit
